## 使用

- #### 安装

  ```
  npm install
  ```

- #### 运行

  ```
  yarn serve
  ```

- #### 打包

  打包前请注意将`.env.production`的`VITE_BASE_PATH`修改为`/`

  ```
  yarn build
  ```
